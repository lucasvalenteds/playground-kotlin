# Playground Kotlin

![jdk] ![kotlin]

This repository contains code written to demonstrate libraries and frameworks targeting Kotlin.

| Description | Command |
|:---|:---|
| List all projects | `./gradlew projects` |
| Run unit tests | `./gradlew test` |
| Run integration tests | `./gradlew integTest` |
| Run e2e tests | `./gradlew e2eTest` |
| Run unit tests of one project | `./gradlew :project:test` |
| Check for newer dependencies | `./gradlew dependencyUpdates` |
| Update all dependencies to latest | `./gradlew useLatestVersion` |
